# CSS Introduction Exercises

This repo contains a series of CSS learning exercises that build foundational skills in styling HTML.

## Exercise summaries

- Exercise 1: Explored CSS specificity and cascade order across inline, internal, and external styles. I learned how inline styles override stylesheet rules and how `!important` can force a rule to win.
- Exercise 2: Found and corrected HTML/CSS syntax mistakes in a simple styled page. I learned that a tiny typo in a `link` tag or a missing semicolon can stop styles from working.
- Exercise 3: Practiced CSS color formats and conversions. I learned how named colors, hex, RGB, and HSL all represent the same colors in different ways, and how alpha adds transparency.
- Exercise 4: Built a press release page with external CSS and Google Fonts. I learned how to import custom fonts, style text, and structure a content-focused page with headings and paragraphs.
- Exercise 5: Worked on specificity, inheritance, and color equivalence. I learned how paragraph styles inherit from `body`, and how later rules override earlier rules when specificity is equal.
- Exercise 6: Created a themed coffee page with typography styling and spacing. I learned how to use Google Fonts for headings and body text, plus how to apply line-height and word-spacing.
- Exercise 7: Reflected on CSS principles and best practices. I learned why HTML provides the structure and CSS provides the decoration, and why external stylesheets are better for larger sites.
