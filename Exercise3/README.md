### 1. Write `color: red;` in three other formats (Hex, RGB, and HSL). All four should look identical or extremely close on screen.
- Hex format: `color: #FF0000;`
- RGB format: `color: rgb(255, 0, 0);`
- HSL format: `color: hsl(0, 100%, 50%);`

### 2. What is the difference between `rgb(255, 0, 0)` and `rgba(255, 0, 0, 0.5)`? Even though `rgba` wasn't covered in the lesson, what do you think the `a` stands for based on context? What visual difference would you expect?
- The `a` stands for alpha, which controls the opacity of the color. `rgba(255, 0, 0, 0.5)` would look like a semi-transparent red, so it would let some of the background show through.

### 3. Convert `#000000` and `#ffffff` to RGB format. What colors are these?
- `#000000` → `rgb(0, 0, 0)` which is black.
- `#ffffff` → `rgb(255, 255, 255)` which is white.
