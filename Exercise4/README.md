# Exercise 4 Summary

This exercise creates a press release page using external CSS and Google Fonts.

I used a large heading, subheading, and multiple paragraphs to structure the content. The exercise taught me how to import fonts from Google, keep CSS in a separate stylesheet, and use typography styles to make text look polished.

### Date line styling
I temporarily added `font-size: 14px` and `text-decoration: line-through` to the date line, which made the date look crossed out and therefore inappropriate for a publication date, so I removed it because a date should appear as current and readable, not canceled.
